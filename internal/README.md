# 内部核心组件

此目录包含WAF的核心内部组件，不对外暴露API：

## 主要子目录

- **api**：API层实现，处理请求和响应
- **config**：配置管理模块
- **middleware**：HTTP中间件，如认证、日志、监控等
- **models**：数据模型定义
- **rules**：规则引擎实现
- **analysis**：行为分析模块
- **ai**：AI判定模块，用于高级威胁检测
- **proxy**：反向代理功能实现
- **logging**：日志系统
- **metrics**：性能指标收集
- **storage**：数据存储接口和实现
- **admin**：管理功能实现
